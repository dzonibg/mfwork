MFwork - Simple PHP MVC Framework.

---

Features:

- ORM
- Auth
- Bootstrap 4 bundled
- MVC

---

Directory structure:

 Framework - files essential to the FW

* /Controllers - Where your controllers go
* /Models - Where your models go and extend the Model class
* /Views - Where your frontend goes.
